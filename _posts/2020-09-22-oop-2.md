# 상속(inheritance)
- 정의
  - 기존 클래스를 재사용하여 새로운 클래스를 작성하는 것
  - 조상클래스를 extend
- 장점
  - 코드의 **재사용성**을 높이고 코드의 **중복을 제거**하여 프로그램의 생산성과 **유지보수** up

상속관계 이외에도 클래스를 재사용하는 방법 : 포함(Composition)관계
```
상속관계 '~은 ~이다.(is-a)'
포함관계 '~은 ~을 가지고 있다.(has-a)'
```
[자바는 다중상속이 불가능하다.](https://ahnyujin.github.io/posts/single-inheritance)
