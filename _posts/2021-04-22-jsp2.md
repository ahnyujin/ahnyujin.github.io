# JSP 내장객체

- JSP request, response 객체
- config 객체
  - 서블릿에서 내장객체를 사용해 초기화 진행
- application 객체
  - 어플리케이션 전체 공유
- out 객체
  - out.print()를 통해 출력 가능
- exception 객체

# DAO와 DTO

 - servlet 기능이 많으므로 데이터에 접근하는 역할을 모듈화하는 객체
 
DTO : Data Transfer Object(=VO?)
 - DB 데이터를 자바형태로 변환하는 객체
