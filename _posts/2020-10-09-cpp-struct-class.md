# 클래스와 구조
- 클래스와 구조체 모두 "고유한 형식을 정의하는 데 사용되는 구문"
- 형식의 상태 및 동작을 설명하는 데 사용되는 (데이터 멤버)와 (멤버 함수)를 포함

# C++ 클래스와 구조체의 차이 = default 접근제어자가 다르다
변수와 함수에 접근제어자를 명시하지 않는 경우 차이가 있다.
- 클래스의 경우 private로 선언되고
- 구조체의 경우 public으로 선언된다
